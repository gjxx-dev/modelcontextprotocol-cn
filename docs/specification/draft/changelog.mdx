本文档列出了自上一修订版[2025-06-18](/specification/2025-06-18)以来对Model Context Protocol (MCP)规范所做的更改。

## 主要更改

1. 通过支持[OpenID Connect Discovery 1.0](https://openid.net/specs/openid-connect-discovery-1_0.html)增强授权服务器发现。(PR [#797](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/797))
2. 允许服务器将图标作为工具、资源、资源模板和提示的附加元数据公开([SEP-973](https://github.com/modelcontextprotocol/modelcontextprotocol/issues/973))。
3. 通过`WWW-Authenticate`增强授权流程的增量范围同意([SEP-835](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/835))
4. 提供工具命名的指导([SEP-986](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/1603))

## 次要更改

1. 澄清服务器必须在Streamable HTTP传输中对无效的Origin标头使用HTTP 403 Forbidden响应。(PR [#1439](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/1439))
2. 更新了[安全最佳实践指导](https://modelcontextprotocol.io/specification/draft/basic/security_best_practices)。
